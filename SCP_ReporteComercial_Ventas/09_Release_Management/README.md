# Release Management

## Objetivo
Controlar la liberación a producción.

## Contenido

- Notas de liberación
- Checklist de despliegue
- Plan de rollback
- Validación en producción

## Documentos esperados

- REL_Release_Notes.docx
- Deployment_Checklist.xlsx
- Rollback_Plan.docx
- Production_Validation_Record.docx

## Buenas Prácticas

- Documentar todos los cambios
- Validar antes y después del despliegue
- Tener plan de reversa

## Regla Clave

Liberar sin control es un riesgo operativo

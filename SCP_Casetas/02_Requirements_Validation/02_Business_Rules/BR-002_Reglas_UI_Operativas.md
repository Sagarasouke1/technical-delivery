# BR-002 — Reglas Operativas de Interfaz

## RN-09 — Confirmación de carga
El usuario debe confirmar antes de procesar archivo

## RN-10 — Selección obligatoria de tipo
Debe seleccionar AEO o FLA

## RN-11 — Aprobación/Rechazo
Cada caseta puede:
- Aprobarse
- Rechazarse con comentario

## RN-12 — Clasificación de viaje
Debe asignarse:
- Cargado
- Pagado
- Vacío autorizado

## RN-13 — Actualización de precios
Debe requerir URL válida

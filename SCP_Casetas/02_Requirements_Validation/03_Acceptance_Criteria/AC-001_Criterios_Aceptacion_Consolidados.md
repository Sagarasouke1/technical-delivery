# AC-001 - Criterios de Aceptación Consolidados

## Importación
- El sistema debe cargar archivos descargados del portal sin modificaciones manuales.
- Debe validar compatibilidad del formato.
- Debe notificar errores de carga.

## Validación
- Debe identificar casetas fuera de ruta o costos incorrectos.
- Debe detectar diferencias de costo.
- Debe identificar duplicados.

## Trazabilidad
- Debe registrar usuario, fecha y acción.
- Debe permitir consulta histórica.
- No debe permitir alteración posterior.

## Reportes
- Debe generar reportes por unidad y viaje.
- Debe incluir tipo de inconsistencia.
- Debe permitir exportación.

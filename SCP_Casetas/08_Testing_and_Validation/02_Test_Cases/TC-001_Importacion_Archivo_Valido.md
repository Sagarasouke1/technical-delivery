# TC-001 - Importación de Archivo Válido

## Objetivo
Validar que el sistema permita importar correctamente un archivo descargado del portal PASE sin modificaciones manuales previas.

---

## Requerimiento Relacionado
- RF-01 Importación de Archivos
- HU relacionada: Importación de archivos / Importación directa de archivos del portal

---

## Precondiciones
- El usuario tiene acceso al sistema.
- El usuario cuenta con un archivo válido descargado del portal PASE.
- El sistema se encuentra disponible.

---

## Datos de Entrada
- Archivo válido del portal PASE

---

## Pasos
1. Ingresar al módulo de importación.
2. Seleccionar el archivo válido.
3. Ejecutar la carga del archivo.
4. Esperar el procesamiento inicial.

---

## Resultado Esperado
- El sistema acepta el archivo.
- El sistema procesa la información sin requerir edición manual previa.
- La estructura base del archivo se conserva.
- La información queda disponible para el siguiente paso del proceso.

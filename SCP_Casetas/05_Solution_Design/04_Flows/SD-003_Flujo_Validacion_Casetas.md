# SD-003 - Flujo de Validación de Casetas

1. El usuario importa archivo del portal PASE.
2. El sistema valida el formato.
3. El sistema estructura información por económico, fecha, caseta, importe, clase vehicular y TAG.
4. El sistema valida por viaje considerando fechas, carta de instrucciones, ruta autorizada y casetas esperadas.
5. El sistema detecta inconsistencias, diferencias de costo y duplicados.
6. El usuario registra incidencias o justificaciones cuando aplique.
7. El sistema genera reporte consolidado.
8. El sistema registra trazabilidad de acciones.

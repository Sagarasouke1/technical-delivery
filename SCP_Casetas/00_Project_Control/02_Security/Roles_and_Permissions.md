# Roles and Permissions

## Rol: Operativo Liquidaciones
- importar archivos
- consultar validaciones
- registrar incidencias
- exportar resultados

## Rol: Supervisor / Administrativo
- consultar reportes
- revisar anomalías
- auditar validaciones
- consultar históricos

## Rol: Operaciones
- justificar incidencias
- validar observaciones operativas
- confirmar o rechazar casos
